# Chinese Academic Review Writing Style

JL 的中文综述写作风格约束。不同于 proposal 写作，综述要求更高的事实密度和更冷的学术语感。

### 标题命名规范

**禁用比喻、拟人、口语化表达。** 标题是结构路标，不是叙事修辞。以下模式在综述标题中不得出现：

| 禁用模式 | 示例 | 替代 |
|---------|------|------|
| 拟人/命运化 | "氧化膜命运的调控因素" | "氧化膜稳定性的调控因素" |
| 开关/关键比喻 | "pO²⁻：氧化膜的总开关" | "pO²⁻对氧化膜稳定性的控制作用" |
| 引号比喻 | 氧化膜存在的"大气" | 直接陈述：熔盐化学环境 |
| 口语化动词 | "看见"氧化膜的行为 | "表征"或"研究方法" |
| 经济比喻 | "工程收益"/"化学代价" | "降低熔点的有利效应"/"加速退化的不利效应" |

### 段落结构

- 每段只承载**一个论点**。3-5句的短段落优于10句以上的长段落。
- 论点句在前，支撑句在后。不把结论藏在段落中间。
- 段落之间不需要平滑过渡——逻辑关系由内容本身传达，不由"此外""另一方面""值得注意的是"传达。
- **首行缩进**：中文正文段落必须设置首行缩进（两字符，~0.74 cm）。标题、图注、参考文献除外。生成 docx 时必须显式设置 `first_line_indent`。

## 禁用表达

以下表达在任何中文综述正文中不得出现：

| 禁用 | 原因 | 替代 |
|------|------|------|
| 随着...的发展 | 万能开头，无信息量 | 直接陈述背景事实 |
| 值得注意的是 | 作者不该替读者决定什么值得注意 | 删除或用数据证明 |
| 毫无疑问 | 科学中没有毫无疑问的事 | 删除或用实验证据等级替代 |
| 众所周知 | 如果不是所有人都知道，就是错误陈述 | 给出引用 |
| 具有重要意义 | 空洞判断 | 说明具体影响了什么 |
| 填补空白 | 除非有证据证明此前无人做过 | 用"尚未见系统报道" |
| 首先/其次/再次/最后 | 机械堆砌感 | 保留逻辑必需的衔接，删公式化套话 |

## 科学语感

- **给数字，不给了形容词。** 不是"显著增加"，而是"+285.7%"。不是"较高浓度"，而是"≥0.135 wt%"。不是"长时间运行"，而是"7000 h"。
- **区分事实等级。** 实验显示/数据表明 ≠ 可能/推测/有待验证。不确定的事不写成确定的口吻。
- **引用的动词要精确。** Smith和Jones[5]**报告**了腐蚀速率（事实）。Chen等[11]**提出**临界阈值0.135 wt%（主张）。Wang等[36]的实验中TC条件腐蚀**低于**等温（反直觉发现）。
- **化学式规范。** 上下标正确：Cr₂O₃不是Cr2O3。化学反应式用⇌表示可逆，→表示不可逆或净方向。离子电荷用上标：O²⁻不是O2-。

## AI味特指模式

以下模式在中文综述中暴露AI写作痕迹，必须避免：

1. **总结前句的开头句。** "上述分析表明..." "综上所述..." — 段落已经结束，不需要再总结。
2. **形容词作填充。** "系统的研究""深入的探讨""全面的分析" — 如果工作本身没有证明系统性，不替作者加帽子。
3. **"不仅...而且..."结构。** 除非两个分句之间存在真正的递进关系，否则是句式模板。大多数时候用两个独立短句更好。
4. **段尾套话。** "这为后续研究提供了参考""该结果具有工程指导意义" — 如果你不能说出具体是什么参考、指导了什么，就不要写。
5. **过度平滑。** 真人学者写中文综述段落之间是有跳跃感的——过渡是内容逻辑给的，不是连接词给的。

## 章节标题规范

中文综述章节标题必须遵循科学严肃性约束：

| 禁用 | 原因 | 替代示例 |
|------|------|---------|
| 比喻词（大气、命运、总开关、看见） | 人文化、口语化，不符合自然科学语域 | "化学环境" 替代 "\"大气\"" |
| 拟人化（氧化膜的命运、材料的寿命故事） | 赋予研究对象主体性，不严肃 | "氧化膜稳定性的调控因素" |
| 引导/引号口语词（如何"看见"） | 标题应是标签，不是提问 | "氧化膜研究方法" |
| 经济学比喻（收益、代价、权衡） | 可用但需中性化 | "双重影响"替代"收益与代价" |
| "总开关"/"钥匙"/"密码" 等比喻 | 标题不是科普文章 | "控制作用"替代"总开关" |

核心原则：**标题是结构的路标，不是叙事的修辞。** 读完标题应该知道这一节讲什么对象+什么角度，不应有任何修辞设计的痕迹。

## 句式偏好

- 主语明确，少用"其""该""此"等代词指代前文概念——容易造成指代歧义。
- 短句优于长句。一个句子超过40字时检查是否可以拆分。
- 被动语态有限使用。中文综述被动语态过多读起来像翻译，主动语态更直接。"实验表明"优于"被实验所表明"。

## 章节标题规范

标题中不得使用口语化比喻。以下为本次综述写作中实际踩过的坑：

| 禁用（比喻/口语） | 替换（科学严肃） |
|:---|:---|
| "氧化膜存在的\"大气\"" | 直接删掉比喻，保留"熔盐化学环境" |
| "氧化膜命运的调控因素" | "氧化膜稳定性的调控因素" |
| "pO²⁻：氧化膜命运的\"总开关\"" | "pO²⁻对氧化膜稳定性的控制作用" |
| "降低熔点的工程收益与加速氧化膜退化的化学代价" | "NaNO₂对氧化膜稳定性的双重影响" |
| "如何\"看见\"氧化膜的行为" | "氧化膜研究方法" |

规则：标题中不出现"命运""总开关""大气""收益/代价""看见"等拟人化或口语化词汇。标题仅陈述内容，不加修饰。

## 文献处理

- 引用给出作者+年份（在正文初稿中用占位符[n]），不是"有研究显示""文献报道"。
- 同一事实有多篇文献支撑时，选最有代表性的1-2篇引用，不堆砌。
- 不替作者说话——不把论文中作者没有明确提出的推论当作论文的结论来引用。
