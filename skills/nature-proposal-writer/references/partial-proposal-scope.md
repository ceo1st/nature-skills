# Partial proposal scope control

Use when JL is writing only one part/stage of a doctoral research plan, not the full proposal.

## Rule

Do not force a partial-stage draft into a full doctoral-plan structure. First lock the stage boundary.

## Scope-lock questions/checks

1. Which part/stage is being written now?
2. Which later stages are intentionally deferred?
3. What missing information blocks later-stage writing? Examples: lab visit, supervisor decision, equipment availability, procurement, platform design.
4. Should deferred content appear only as “后续研究方向” rather than as current objectives or methods?

## For JL's chloride-salt proposal class

If the current text is the first part of the NaCl-KCl-MgCl2-ZnCl2 plan:

- Current scope: candidate composition screening, offline salt chemistry/thermal treatment, Zn/ZnO or related purification route evaluation, purified-salt static immersion corrosion verification.
- Deferred scope: online sampling, three-electrode electrochemistry, flange/platform design, detailed lab-platform engineering.
- Reason: these later parts should wait until JL visits the Beijing laboratory and inspects available experimental conditions.
- Writing consequence: keep later platform/electrochemistry content as brief follow-on direction; do not expand it into current-stage objectives or methods.

## Common pitfall

Bad: “This draft lacks a complete doctoral-plan platform/electrochemistry design, so expand it now.”

Good: “This is a first-stage plan; evaluate whether the first-stage offline loop is coherent, executable, and sufficient to justify later platform work.”
